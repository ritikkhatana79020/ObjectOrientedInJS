# this keyword
*Functions in Javascript are essentially objects. Likes objects they can be assigned to variables, passed to other fucntions and returned from function*
*The value that 'this' stores is the current execution context of the JS program. Thus, when used inside a function 'this' value will change depending on how that function is defined, how it is invoked and the default execution context.*

*When you just ``` console.log(this) ``` is global context. You would see the 'window' object being logged into the cosole*
